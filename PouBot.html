PouBot

Un programme Python capable de jouer aux mini-jeux du jeu Pou

Description du projet

PouBot est un projet personnel réalisé en terminale. C'est un programme Python qui utilise la bibliothèque PyAutoGUI pour automatiser les mini-jeux du jeu Pou.

Fonctionnalités

  1. Détection automatique de la position du jeu sur l'écran
  2. Analyse en temps réel des éléments du jeu
  3. Prise de décision et exécution des actions appropriées
  4. Support de plusieurs mini-jeux différents

Technologies utilisées

  • Python
  • PyAutoGUI pour l'automatisation
  • OpenCV pour la détection d'images
  • NumPy pour les calculs

Défis et solutions

Le principal défi était de rendre le bot suffisamment rapide pour réagir en temps réel aux événements du jeu. J'ai optimisé les algorithmes de détection et utilisé des techniques de mise en cache pour améliorer les performances.

Résultats

Le bot est capable de jouer de manière autonome et d'obtenir des scores élevés dans plusieurs mini-jeux. Ce projet m'a permis d'approfondir mes connaissances en Python et en vision par ordinateur.
